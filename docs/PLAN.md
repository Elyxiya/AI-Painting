# AI-Painting 开发计划

> **版本策略**：砍掉的功能不丢弃，全部移入 v2 路线图。v1 聚焦核心价值，v2 逐步完善体验。

---

## 版本划分

| 版本 | 目标 | 预期周期 | 交付内容 |
|------|------|----------|----------|
| **v1.0** | MVP（最小可行产品） | **1-2 天** | 基础画布 + 语音转文字 + 简单命令解析 |
| **v2.0** | 增强体验 | **2-3 天** | LLM 意图理解 + 唤醒词检测 + 撤销历史 + 导出优化 |
| **v3.0** | 协作与稳定 | **3-4 天** | 多窗口同步 + 版本回滚 + 大画布优化 |

---

## v1.0 MVP 范围

### 交付内容

| 模块 | 功能 | 状态 |
|------|------|------|
| **画布** | Konva 画布 + 7 种绘图工具 | v1 |
| **图层** | 多图层、显示/隐藏、锁层 | v1 |
| **语音** | Whisper WASM 转文字 + PTT 按钮 | v1 |
| **命令** | 正则规则引擎（简单命令解析） | v1 |
| **保存** | JSON 项目文件 + 自动保存 | v1 |
| **打包** | Electron 打包为 .exe | v1 |

### v1 砍掉的功能（移入 v2）

| 原计划功能 | v1 砍掉原因 | 移入版本 |
|-----------|-------------|---------|
| LLM 意图理解（Ollama） | Windows 兼容性问题，耗 3-4h | v2 |
| Porcupine 唤醒词检测 | WASM + Electron 兼容性坑 | v2 |
| 撤销/重做历史 | 节省 2h 开发时间 | v2 |
| Web Speech API 降级链路 | v1 直接用 Whisper WASM | v2 |
| 版本回滚/备份 | 节省 2h | v3 |
| 多窗口状态同步 | 节省 3h | v3 |
| LOD/视口裁剪优化 | P1 阶段处理 | v3 |
| 撤销历史压缩 | P1 阶段处理 | v3 |

## Agent 并行策略（ v1 开发）

```
Agent-A: 项目脚手架 + Konva 画布 + 图层系统

Agent-C: 文件保存 + Electron 打包

Human: 关键节点验收
```

**预计有效 agent 工作时间：3-4h（1-2 天内完成 v1）**

## 关键决策（已确认）

| 决策 | 选择 | 原因 |
|------|------|------|
| LLM 方案 | **DeepSeek API 云端** | 免费额度，无需 Ollama 安装，无 Windows 兼容性坑 |
| Whisper 模型 | **base**（74MB） | 准确率优先，v1 体验更重要 |
| 撤销历史 | **50 步** | 够用，不占用过多内存 |

> **备注**：v1 阶段暂不接入 LLM，降级为正则规则引擎。v2 通过环境变量 `DEEPSEEK_API_KEY` 接入 DeepSeek API。
